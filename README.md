# Mackiavelli — Productized AI Systems that Work

Static one-page site built with Tailwind Play CDN.

## Local preview
Just open `index.html` in your browser.

## Deploy on GitHub Pages
1. Create a new public repo (e.g. `mackiavelli-site`) and upload all files.
2. In **Settings → Pages**, set **Branch: `main`** and **Folder: `/ (root)`**, then **Save**.
3. Your site will appear at `https://<you>.github.io/mackiavelli-site/`.

## Customize
- Replace `assets/logo.svg`, `assets/favicon.ico`, and `assets/og-image.jpg`.
- Update social links and `og:url` in `<head>`.
- Swap the CTA form `alert(...)` for a real endpoint (Formspree/Netlify/your backend).

## Optional
- Add `CNAME` with your custom domain.
- Add analytics (e.g., Plausible).
